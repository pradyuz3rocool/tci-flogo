     1	# flogo-contrib
     2	
     3	[![Build Status](https://travis-ci.org/TIBCOSoftware/flogo-contrib.svg?branch=master)](https://travis-ci.org/TIBCOSoftware/flogo-contrib.svg?branch=master)
     4	
     5	Collection of Flogo activities, triggers and models.
     6	
     7	## Contributions
     8	
     9	### Activities
    10	* [awsiot](activity/awsiot): Aws IOT shadow update
    11	* [coap](activity/coap): CoAP messaging 
    12	* [counter](activity/counter): Global counter  
    13	* [log](activity/log): Simple flow Logger 
    14	* [rest](activity/rest): Simple REST invoker
    15	* [twilio](activity/twilio): Simple Twilio SMS sender
    16	* [websocket] (activity/wsmessage): Simple Websocket Message
    17	
    18	### Triggers
    19	* [coap](trigger/coap): Start flow via CoAP
    20	* [mqtt](trigger/mqtt): Start flow via MQTT
    21	* [rest](trigger/rest): Start flow via REST
    22	* [timer](trigger/timer): Start flow via Timer
    23	 
    24	### Models
    25	* [simple](model/simple): Basic flow model
    26	
    27	## Installation
    28	
    29	#### Install Activity
    30	Example: install **log** activity
    31	
    32	```bash
    33	flogo install github.com/TIBCOSoftware/flogo-contrib/activity/log
    34	```
    35	#### Install Trigger
    36	Example: install **rest** trigger
    37	
    38	```bash
    39	flogo install github.com/TIBCOSoftware/flogo-contrib/trigger/rest

    41	#### Install Model
    42	Example: install **simple** model

    44	```bash

    48	## Contributing and support
    49	
    50	### Contributing
    51	
    52	New activites, triggers and models are welcome. If you would like to submit one, contact us via [Slack](https://tibco-cloud.slack.com/messages/flogo-general/).  Contributions should follow these guidelines:
    53	
    54	* Code must be [gofmt](https://golang.org/cmd/gofmt/) compliant.
    55	* Execute [golint](https://github.com/golang/lint) on your code.
    56	* Document all funcs, structs and types.
    57	* Ensure that 'go test' succeeds.
