     6	# Return
     7	This activity allows you to reply to a trigger invocation and map output values. After replying to the trigger, the flow ends (this will be the last actvity in your flow).
     8	
     9	## Installation
    10	### Flogo Web
    11	This activity comes out of the box with the Flogo Web UI
    12	### Flogo CLI
    13	```bash
    14	flogo install github.com/TIBCOSoftware/flogo-contrib/activity/actreply
    15	```
    16	
    17	## Schema
    18	Input and Output:
    19	
    20	```json
    21	{
    22	  "input":[
    23	    {
    24	      "name": "mappings",
    25	      "type": "array",
    26	      "required": true,
    27	      "display": {
    28	        "name": "Mapper",
    29	        "type": "mapper",
    30	        "mapperOutputScope" : "action.output"
    31	      }
    32	    }
    33	  ],
    34	  "output": [
    35	  ]
    36	}
    37	```
    38	
    39	## Settings
    40	| Setting     | Required | Description |
    41	|:------------|:---------|:------------|
    42	| mappings    | True     | An array of mapping that are executed when the activity runs |
    43	
    44	
    45	## Example
    46	The below example allows you to configure the activity to reply and set the output values to literals "name" (a string) and 2 (an integer).
    47	
    48	```json
    49	{
    50	  "id": "actreturn_5",
    51	  "name": "Return",
    52	  "description": "Simple Return Activity",
    53	  "activity": {
    54	    "ref": "github.com/TIBCOSoftware/flogo-contrib/activity/actreturn",
    55	    "input": {
    56	  	"mappings":[
    57	      { "type": "literal", "value": "name", "mapTo": "Output1" },
    58	      { "type": "literal", "value": 2, "mapTo": "Output2" }
    59	    ]
    60	  }
    61	}
    62	```
